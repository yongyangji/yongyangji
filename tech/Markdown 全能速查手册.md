# 🚀 Markdown 语法全能速查手册

这是一份专为开发者、科研人员和创作者准备的 Markdown 语法参考手册。你可以将它直接作为你的第一个 Markdown 文档进行测试和参考。

---

## 1. 基础排版 (Basics)

### 标题
使用 `#` 号表示，支持六级标题。
# 一级标题
## 二级标题
### 三级标题
#### 四级标题

### 文本样式
* **加粗**：`**加粗文本**`
* *斜体*：`*斜体文本*`
* ***粗斜体***：`***粗斜体***`
* ~~删除线~~：`~~删除线~~`
* <u>下划线</u>：`<u>下划线</u>` (HTML 标签支持)

## 3. 代码与引用 (Code & Quotes)

### 引用块
> 这是一段引用。
>> 这是一段嵌套引用。

### 代码演示
**行内代码**：使用 `print("Hello World")` 打印。

**多行代码块**：
```python
def main():
    # 这是一个带有语法高亮的 Python 代码块
    msg = "Markdown is simple but powerful!"
    print(msg)

if __name__ == "__main__":
    main()
```

## 4. 表格 (Tables)

| 功能 | 语法示例 | 推荐程度 |
| :--- | :---: | ---: |
| 标题 | `# Heading` | ⭐⭐⭐⭐⭐ |
| 加粗 | `**Bold**` | ⭐⭐⭐⭐⭐ |
| 公式 | `$E=mc^2$` | ⭐⭐⭐⭐ |

> **提示**：冒号 `:` 的位置决定对齐方式（左、中、右）。

## 6. 数学公式 (LaTeX)

Markdown 支持高质量的 LaTeX 数学排版：

* **行内公式**：例如质能方程 $E = mc^2$。
* **块级公式**：
$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

## 7. 其他常用符号
* **分割线**：使用 `---` 或 `***`
* **转义字符**：如果想输出 `#` 本身而不是标题，使用 `\#`
